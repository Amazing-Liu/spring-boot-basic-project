# 本项目主要演示了springboot与thymeleaf的使用
主要内容在index.html中,建议在浏览器中查看  
一些thymeleaf的配置在yml中,都有注解

## thymeleaf的依赖
```xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-thymeleaf</artifactId>
</dependency>
```
